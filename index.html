Eyeball Generator

Description:

The eyeball generator I made is a generative web application which allows the user to create a trail of eyes which follow along with their mouse and change color randomly as it moves. Along with this, I was able to create two controls on the top right where the user can change the height and width of the eyes.

Parameters:

Width - Change the width of the eye
Height- Change the height of the eye
